# MangoAkinator (Лабораторная работа по дисциплине Системы искусственного интеллекта №1)
## Что представляет из себя проект?
***MangoAkinator*** &mdash; это знакомая всем игра по отгадыванию чего-либо, однако в данном случае темой является манга. Вы можете отгадывать существующую мангу, добавлять новую мангу, анализировать полученный ответ, просматривать базу знаний, просматривать отдельные элементы базы знаний. **Проект представляет из себя простенькую консольную игру. Действия производятся с помощью консольного ввода.**
## Запуск игры
Игру можно запустить файлом MangoAkinator.exe.
## База знаний 
База знаний находится в файле Knowledge.json **(если так получилось, что у вас нет этого файла при запуске исполняемого файла, он создастся с начальными данными в той же директории, что и исполняемый файл)**.
